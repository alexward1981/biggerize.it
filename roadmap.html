Biggerize.it

There are a lot of things I still plan to do with this app. Both in terms of adding new features and fixing issues. If you are interested, keep reading:

Issues
Google fonts not loading in Mobile browsers
Text resizing still isn't perfect (Can look too small/big depending on screen-size)
The site still isn't 100% mobile friendly.
Text gets lost on back button.
Features
Social Sharing - Share your biggerized text with others without having to copy the url
Text effects - The Google fonts API provides some truly ghastly effects, I plan on making them available here
Random text - Can't think what to write? Just hit random (Writing pointless features is 1/2 the fun!)
Adverts - Sorry, we all need to make money somehow, it will just be one banner add on the homepage, that's all, I promise.
Emoticons - I want to auto convert emoticon characters into actual emoticons
Backgrounds - I want to add a selection of backgrounds you can add to your biggerised text
Tips - It's possible to play around with query strings to get a different output, I want to add a tips section to show people how.
Previews - I want to make it so you get a mini preview of your font choice on the homepage
More fonts - Google fonts have a lot of options, so far I've only added 5, I plan to add lots more.
Multiple lines - At the moment only one line can be added, sometimes it's not enough so an option to add another line with it's own style would be ace.
Other stuff - The site went live because it reached a stage where it was 'good enough', however I'm far from finished, keep checking back here to see if I add even more features to the roadmap!

If you happen to find any more bugs or have any feature requests, please tweet them to me and I'll add them to the list!
